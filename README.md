# alfred-gpt

## How to use:

1. **You must add an API KEY** to config.json:

```
{
	"API_KEY" : "YOUR_API_KEY_GOES_HERE",
	"MODEL" : "text-davinci-003",
	"TEMPERATURE" : 0.0,
	"MAX_TOKENS" : 1000
}
```
